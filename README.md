## Deployment

This repo is connected to **Cloudflare Pages**. Every push to `main` automatically deploys to production.

```bash
# Make your changes, then:
git add .
git commit -m "your message"
git push origin main
```

Cloudflare Pages will deploy within ~1 minute.
